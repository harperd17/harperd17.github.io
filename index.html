Dashboard for Arizona Soil Area Characteristics and Flash Floods

David Harper

Instructions:

1. Select which region to look at and which variable to be displayed.

 a. Select region by using the area selection drop down.

 b. Click on a region on the map.

 c. Select variable using the variable selection drop down.

2. Zoom into a selected region.

 a. Use the zoom in / zoom out button.

 b. Click on the selected region.

 c. Double click on a region when initially selecting.

3. View region and area information.

 a. When hovering over an Arizona region, a text window will pop up showing the region name and the number of flash floods since August 2017.

 b. When hovering over a soil area while zoomed into a selected region, a text window will pop up showing the soil area name.
